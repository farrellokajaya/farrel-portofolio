## Getting Started

### Prerequisites

Make sure Node.js and npm are installed on your system.

### Installation

Clone the repository:

```bash
git clone https://github.com/farrellokajaya/farrel-portofolio.git
```

Enter the project directory:

```bash
cd farrel-portofolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Starts the local development server |
| `npm run lint` | Runs ESLint checks |
| `npm run typecheck` | Checks TypeScript types without generating files |
| `npm run build` | Creates a production build |
| `npm run start` | Runs the production server |
| `npm run check` | Runs lint, type checking, and production build |

## Quality Checks

Before committing changes, run:

```bash
npm run check
```

This command verifies:

1. ESLint rules
2. TypeScript type safety
3. Production build readiness
